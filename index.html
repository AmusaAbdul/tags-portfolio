HTML TAGS AND THEIR USES

  1. !DOCTYPE: It defines the document type.
  2. a: It defines a hyperlink.
  3. abbr: It defines an abbreviation or an acronym.
  4. address: It defines contact information for the author/owner of a document.
  5. area: It defines an area inside an image map.
  6. article: It defines an article.
  7. aside: It defines content aside from the page content.
  8. audio: It defines embedded sound content.
  9. b: It defines bold text.
  10. base: It specifies the base URL/target for all relative URLs in a document.
  11. bdi: It isolates a part of text that might be formatted in a different direction from other text outside it.
  12. bdo: It overrides the current text direction.
  13. blockquote: It defines a section that is quoted from another source.
  14. body: It defines the document's body.
  15. br: It defines a single line break.
  16. button: It defines a clickable button.
  17. canvas: It used to draw graphics, on the fly, via scripting (usually JavaScript).
  18. caption: It defines a table caption.
  19. center: It defines centered text.
  20. cite: It defines the title of a work.
  21. code: It define a piece of computer code.
  22. col: It specifies column properties for each column within a colgroup element.
  23. data: It adds a machine-readable translation of a given content.
  24. datalist: It specifies a list of pre-defined options for input control.
  25. dd: It defines a description/value of a term in a description list.
  26. del: It defines the text that has been deleted from a document.
  27. details: It defines additional details that the user can view or hide.
  28. dfn: It specifies a term that is going to be defined within a content
  29. dialog: It defines a dialog box or window.
  30. dir: It defines a directory list.
  31. div: It defines a section in a document.
  32. dl: It defines a description list.
  33. dt: It defines a term/name in a description list.
  34. em: It defines an emphasized text.
  35. embed: It defines a container for an external application.
  36. fieldset: It groups related element in a form.
  37. figcaption: It defines a caption for figure element.
  38. figure: It specifies self-contained content.
  39. font: It defines font, colour, and size for text.
  40. footer: It defines a fooster for a document or section.
  41. form: It defines an HTML form for user input.
  42. frame: It defines a window (a frame) in a frameset.
  43. frameset: It defines a set of frame.
  44. h1 to h6: It defines HTML headings.
  45. head: It containes metadata/information for the document.
  46. header: It defines a header for a document.
  47. hgroup: It defines a header and related content.
  48. hr: It defines a thematic change in the content.
  49. html: It defines the root of an HTML document.
  50. i: It defines a part of text in an alternate voice or mood.
  51. iframe: It defines an inline frame.
  52. img: It defines an image.
  53. input: It defines an input control.
  54. ins: It defines a text that has been inserted into a document.
  55. kbd: It defines keyboard input.
  56. label: It defines a label for an input element.
  57. legend: It defines a caption for a fieldset element.
  58. li: It defines a list item.
  59. linl: It defines the relationship between a document and an external resource (most used to link to stle sheets).
  60. main: It specifies the main content of a document.
  61. map: It defines an image map.
  62. mark: It defines marked/highlighted text.
  63. menu: It defines an unordered list.
    >
  64. meta: It defines metadata about an HTML document.
  65. meter: It defines a scaler measurement within a known range (a guage).
  66. noframes: It defines an alternate content for users that do not support client-side scripts.
  67. noscript: It defines an alternate content for users that do not support frames.
  68. object: it defines a container for an external application.
  69. ol: It defines an ordered list.
  70. optgroup: It defines a group of related options in a drop-down list.
